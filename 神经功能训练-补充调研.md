## 三、第一部分：节拍器与节奏性运动训练（RMT）

### 3.1 什么是节奏性运动训练？

节奏性运动训练（Rhythmic Movement Training, RMT）是一种基于婴儿自然发育动作（摇摆、滚动、爬行）的神经发育训练方法。它有两个主流分支：**Blomberg RMT (BRMT)** 和 **RMT International (RMTi)**。

**核心原理：**
- 婴儿在发育过程中自然做出的节奏性动作（如摇晃、摇摆）是大脑神经通路成熟的关键刺激
- 这些动作帮助原始反射的整合和脑干、小脑的发育
- 有节奏、有规律的重复动作能够激活小脑-前额叶环路，改善时序处理、注意力和冲动控制

**研究基础：**
- 2025年Hirose等人的研究发现，12周的节奏性、平衡和协调运动训练显著减少了ASD/ADHD儿童的ATNR残留（PMC12384944）
- 新西蘭教育部的调研显示，每周使用RMT 4次以上的学生在阅读、社交情绪方面有统计学显著改善（Bu Windsor, 2020）
- Noreika等人2013年的研究证实，ADHD儿童在节拍同步方面存在显著困难，而这种能力可以通过训练改善

### 3.2 节拍器踏步（Metronome Stepping）← 重点补充！

**这就是你昨晚让茂懿做的54拍踏步练习。** 它的学名是「节拍器同步踏步训练」，是RMT和Interactive Metronome体系的核心基础训练。

**训练原理：**
节拍器提供稳定的听觉节奏（通常每分钟54-72拍），孩子需要将踏步动作精确同步到节拍上。这个过程：
- 强制小脑进行精确的时间计算
- 训练前庭系统与听觉系统的协同
- 激活小脑-前额叶环路，改善注意力
- 提供即时的神经反馈（成功同步或脱节）

**标准训练步骤：**

1. **初始阶段（第1-2周）：**
   - 设置节拍器 54 BPM
   - 站立，双脚与肩同宽
   - 跟着节拍左右脚交替踏步（抬脚高度约5-10cm）
   - 目标是让踏步落地的声音和节拍完全重合
   - 每次3分钟，每天1-2次

2. **进阶阶段1（第3-4周）：**
   - 保持54 BPM
   - 踏步同时加手臂摆动：左脚踏步时摆右臂，右脚踏步时摆左臂（对侧模式）
   - 每次5分钟

3. **进阶阶段2（第5-8周）：**
   - 速度变化：54 → 60 → 72 → 60 → 54 BPM 循环
   - 踏步同时增加认知任务：计数、背诗、减法
   - 闭眼踏步（去除视觉补偿，强化前庭-本体感受整合）
   - 踏步时同时转头看左右侧目标

4. **高级阶段（第8周+）：**
   - 在不同表面踏步（硬地→地毯→软垫）
   - 踏步时同时做上肢动作（接抛球、拍手）
   - 单脚节拍踏步（左右脚各30秒交替）

**为什么54 BPM是标准起始速度：**
54 BPM（每分钟54拍）对应于大约1.1秒一次踏步。这个节奏与人类的自然步频接近，不会过快导致压力反应，也不会过慢导致注意力涣散。它是BRMT体系中最常用的起始速度。

### 3.3 Blomberg RMT 核心动作（全部未被上一轮调研覆盖）

以下动作是BRMT体系特有的节奏性运动（区别于上一轮已有的反射整合动作）：

#### 动作1：摇篮式摇摆（Rocking）

**目的：** 激活前庭系统和脑干，模仿婴儿在子宫内和出生后的自然摇摆

**步骤：**
1. 坐在地上，膝盖弯曲，双手抱住膝盖下方
2. 身体向后滚动到肩胛骨位置（不要滚到头部）
3. 利用动量向前滚回坐姿
4. 节奏：向前滚吸气，向后滚呼气
5. 重复15-20次，每天1组

**适用反射：** TLR、Spinal Galant

#### 动作2：四点跪姿前后摇摆

**目的：** 建立上下半身的协调分离，激活小脑

**步骤：**
1. 手和膝盖着地（四足支撑位）
2. 前后摇摆身体：向前时重心移到手上，向后时重心移到脚跟上
3. 幅度不必过大，重点是节奏的稳定性
4. 配合节拍器（40-50 BPM）
5. 每次2-3分钟

**适用反射：** STNR、ATNR

#### 动作3：仰卧腿部节奏性弯曲伸展

**目的：** 激活脊髓格兰特反射整合，改善坐姿稳定性

**步骤：**
1. 仰卧，双腿伸直
2. 缓慢交替弯曲和伸展一条腿：脚后跟沿地面滑向臀部再滑回
3. 左右腿交替
4. 配合节拍器（约40 BPM）
5. 每条腿10次

#### 动作4：侧躺头部转向（配合节拍）

**目的：** ATNR整合变体，强化颈部与眼部运动的分离

**步骤：**
1. 侧躺，双腿微曲
2. 缓慢将头转向天花板方向，停顿2秒后转回
3. 左右侧各5-8次
4. 配合呼吸：转头时呼气，回正时吸气

#### 动作5：坐姿前后摆动（上身）

**目的：** 激活前庭系统的耳石器（utricle和saccule）

**步骤：**
1. 盘腿坐或椅子坐直
2. 上半身缓慢向后倾斜30°再回到垂直
3. 节奏：后倒呼气，回正吸气
4. 然后再向前倾斜30°
5. 重复8-10次一组

### 3.4 Interactive Metronome（IM）体系

Interactive Metronome 是 RMT 的数字化高级版本，由 Dr. Jonathan Vestal（该课程讲师之一）专门认证使用。

**IM 的核心特征：**
- 通过计算机生成精确的听觉节拍
- 孩子需要用手或脚触发传感器来同步节拍
- 系统提供毫秒级的延迟反馈——告诉你提前了多少毫秒或延迟了多少毫秒
- 13个标准任务：分为手部任务、脚部任务和手脚协调任务
- 通常在OT/PT环境下使用，每周3次，每次1小时，共12-18次

**家庭简易版（无需设备）：**
虽然没有IM设备，但可以用节拍器App（手机即可）模拟以下IM核心训练：

1. **拍手同步：** 设置节拍器60 BPM，跟着节拍拍手，目标是拍手声与节拍完全重合
2. **手指敲击：** 设置节拍器54 BPM，用食指交替敲击桌面（左-右-左-右）
3. **踏步拍手组合：** 踏步在节拍上，每4拍拍一次手（1-2-3-拍！1-2-3-拍！）
4. **手脚交替：** 4拍踏步 + 2拍拍手交替循环

**参考研究：**
- Shaffer et al. (2001)：56名ADHD男孩，15小时IM训练后，注意力和运动控制显著改善（AOTA研究）
- Son (2021)：16名ADHD儿童，8周IM训练后阅读流畅性和视觉注意力显著提升
- PMC5890222：IM有效改善ADHD儿童的时序、注意力、工作记忆和处理速度

### 4.2 辐辏训练（Convergence Training）— Pencil Push-ups 及其进阶

**什么是辐辏？**
辐辏是双眼同时向内旋转以聚焦于近处物体的能力。辐辏不足（Convergence Insufficiency, CI）意味着双眼无法有效地协同内转，导致近距离工作时出现复视、眼疲劳、头痛、阅读跳行。

**研究证据：**
- ADHD儿童中CI的发生率是普通儿童的3倍（Granet et al., 2005）
- CITT研究（Convergence Insufficiency Treatment Trial）显示：接受视觉治疗后，CI症状评分从32.1降至9.5（显著改善）
- PMC12531088 (2025)：Pencil push-ups 确实能改善辐辏能力，但**更有效的是结合专业视觉治疗**

#### Pencil Push-ups（铅笔俯卧撑）

**你提到的这个训练，完整方案如下：**

**所需工具：** 一支带字母的铅笔（或任何细长物体上贴一个小贴纸）

**标准步骤：**
1. 坐直，将铅笔水平握在眼前一臂距离
2. 聚焦于铅笔上的字母/贴纸，**保持它清晰且单一**（不能出现重影）
3. 缓慢将铅笔向鼻尖移动
4. 一旦出现重影或模糊，立即停止
5. 保持在这个位置上3-5秒，努力恢复单一清晰视觉
6. 如果不能恢复，就将铅笔稍微后退
7. 再次缓慢向前推
8. 重复10次为一组，每天3-5组

**关键原则：**
- **质量 > 数量：** 出现重影时停下来，不要强迫
- **保持聚焦：** 整个过程视线不能离开目标
- **记录距离：** 记录铅笔距离鼻尖多远时出现重影，每周对比进步

**进阶方案：**
1. **基础版：** 单目标铅笔
2. **进阶1：** 使用带小字的标签（增加视觉难度）
3. **进阶2：** 用细绳串3颗珠子（Brock string的简易版），在不同距离聚焦切换
4. **进阶3：** 推到最近点后，保持3秒，然后缓慢拉回
5. **进阶4：** 结合节拍器——在节拍点上推进和拉回

#### Brock String（布洛克绳）

**这是视觉治疗中最常用的工具之一，比单纯pencil push-ups更有效。**

**所需工具：** 一条长约1-2米的绳子，3颗颜色不同的珠子

**步骤：**
1. 绳子一端固定（如挂在门把手上），另一端拉到鼻尖
2. 3颗珠子分别放在：距鼻子15cm处、臂长处、两者中间
3. 聚焦于近处珠子：你应该看到一颗珠子、两条绳子（X形交叉在珠子上）
4. 切换聚焦到中间珠子：交叉点应该移动到中间珠子
5. 切换聚焦到远处珠子：交叉点移动到远端珠子
6. 每次聚焦保持5-10秒
7. 每天做3轮完整的近-中-远-中-近切换

**为什么有效：**
- 提供视觉反馈（是否看到X形交叉点）
- 训练灵活切换能力（近-远切换）——这是阅读时看黑板-看课本的关键技能
- 强化视觉-前庭整合

### 4.3 调节训练（Accommodation Training）

**什么是调节？**
调节是眼睛改变晶状体形状以聚焦不同距离物体的能力。调节不足导致从看远处切换到看近处时模糊、眼疲劳。

**调节训练项目（未在上一轮覆盖）：**

#### 远近聚焦切换（Hart Chart 的基本原理）

**工具：** 两张纸——一张在远处（3-5米）贴大字母，一张在近处（30-40cm）贴小字母

**步骤：**
1. 先看远处的大字母，大声读出来（1秒）
2. 立即切换到近处的小字母，大声读出来（1秒）
3. 交替进行，每次切换不眨眼
4. 连续做20次切换（远-近为一次）
5. 每天1-2组

**变体（更有效的版本）：**
- 在远处贴一串字母/数字，近处贴另一串
- 远：读第1个字母 → 近：读第1个字母 → 远：第2个 → 近：第2个
- 训练视线的快速精确移动 + 调节

#### 调节灵活性训练（Flipper Lens 的家庭替代）

**工具：** 手机闪光灯或手电筒

**步骤：**
1. 在暗处，看远处的一个目标
2. 手电筒快速照向眼睛（2-3秒），然后移开
3. 瞳孔放大-缩小的快速切换模拟了调节系统的快速响应
4. 注意：这只是一个家庭近似版本，真正的flipper训练需要专业透镜

### 4.4 Saccade（快速眼跳）训练

**什么是 saccade？**
Saccade 是眼睛在两个固定点之间的快速跳跃式移动——这正是阅读时眼睛所做的动作。ADHD儿童的saccade控制通常不成熟，表现为：
- 阅读时跳行、漏字
- 难以从一行结尾准确跳到下一行开头
- 眼球在目标之间移动时过度或不足

**训练项目（未在上一轮覆盖）：**

#### 双目标 Saccade 练习

**工具：** 两张纸，左右各贴一个目标（字母、数字或符号），间隔30-50cm

**步骤：**
1. 坐直，头部保持不动
2. 快速从左边目标跳到右边目标（只动眼睛，不动头）
3. 再快速跳回
4. 目标是每个方向在0.2秒内完成
5. 重复10次为一组
6. **进阶：** 目标是带颜色或字母，如左A右B，读"A"→看右，读"B"→看左

#### 序列 Saccade 练习

**步骤：**
1. 在白纸上贴5-7个点，排成一条水平线，间隔5-8cm
2. 从第1个点跳到第3个点，再跳回第2个点，再到第4个点…（非顺序跳跃）
3. 或沿着顺序依次跳到最后一个，再从最后一个跳回第一个
4. 速度优先，但准确性也要保证（跳过头或没跳到都不算）

#### 阅读模拟 Saccade 练习

**工具：** 一段文字，每一行的第一个字和最后一个字用亮色标注

**步骤：**
1. 从第一行第一个字→快速扫到本行最后一个字→跳到下一行第一个字
2. 不要读内容，只做眼跳练习
3. 目标是每次换行时眼睛能精准落在下一行的起始位置

## 五、第三部分：Dr. Michael Longyear 的脑优化训练体系

### 5.1 Dr. Michael Longyear 是谁？

**Dr. Michael Longyear, DC, DACNB, CCSP**
- 认证脊骨神经科神经学家（Board-Certified Chiropractic Neurologist）
- 曾担任 Life University 神经生命研究所（NeuroLIFE Institute）主任
- 与 Dr. Michael W. Hall 共同教授功能神经学系列课程
- 在佛罗里达州杰克逊维尔运营 Brain Optimization Institute
- 是该课程讲师团队中**中国课程**的对接讲师之一

**他的临床理念（从公开资料提炼）：**
- 当小脑张力改善时，前额叶功能也改善
- 深蹲、跳跃、运动、蹦床等可以改善小脑功能，从而改善前额叶功能
- 强调神经调整（Chiropractic Adjustment）与神经运动训练的结合
- 使用「功能神经学评估」精确诊断神经系统的哪一部分功能不足
- 针对不同半球功能差异提供侧化训练（Hemisphere-Specific Training）

### 5.2 Dr. Longyear 体系的训练特点

从他的临床背景和出版物来看，他的训练体系包括以下几个独特部分：

#### 小脑激活训练（Cerebellar Activation）

**核心理念：** 小脑是大脑的「协调中心」，通过有节奏、精确计时的运动来激活它。

**具体训练（补充上一轮的不足）：**

1. **交替拍膝（有节奏版本）：**
   - 坐姿，节拍器60BPM
   - 左手拍右膝，右手拍左膝（交叉模式）
   - 严格按节拍拍打
   - 进阶：闭眼做

2. **跳床上的认知任务：**
   - 在蹦床上弹跳的同时完成认知任务
   - 弹跳时背乘法口诀、计数、或描述一幅画
   - 这就是「双重任务训练」——激活小脑同时激活前额叶

3. **单侧化训练（Hemisphere-Specific）：**
   - 如果评估发现右侧大脑功能较弱，增加左侧身体的训练
   - 左侧单脚跳跃、左手精细动作、左侧视野的视觉刺激
   - 这基于功能神经学中的「对侧支配」原理

#### 感觉刺激训练（Sensory Stimulation）

Dr. Longyear 体系中的感觉刺激比前一份调研文档更系统：

1. **交替温度刺激：**
   - 交替使用温水和凉水冲洗前臂/小腿
   - 每次30秒，交替3轮
   - 激活皮肤温度感受器→刺激丘脑→促进感觉整合

2. **刷子刺激（Brushing）：**
   - 用软刷子（或粗糙毛巾）按特定顺序刷四肢
   - 顺序：手臂外侧→内侧，腿部外侧→内侧
   - 每次每个部位3-5秒
   - 这个训练实际上来自Wilbarger感觉统合方案

3. **震动刺激：**
   - 使用按摩器或震动玩具在手掌、脚底、脊柱两侧轻按
   - 每个部位10-15秒
   - 激活本体感受器和触觉感受器

#### 前庭-眼球运动整合（Dr. Longyear 体系的特色）

1. **头部固定-眼球运动：**
   - 坐直，头部固定不动
   - 眼球追随一个移动的目标（手电筒、手指、球）
   - 目标做8字形、Z字形、圆形运动
   - 头部不动——强制前庭-眼球反射（VOR）被抑制

2. **眼球固定-头部运动：**
   - 注视一个固定目标（如墙上的贴纸）
   - 缓慢左右转动头部
   - 眼睛保持锁定目标不动
   - 这是真正的VOR训练——在头部运动时保持视觉稳定

3. **身体运动-眼球固定：**
   - 单脚站立，注视固定目标
   - 慢慢身体前后倾斜（从脚踝开始）
   - 眼睛保持注视
   - 这是最难的版本——前庭+平衡+视觉的同步训练

## 六、第四部分：Dr. Jonathan Vestal 的 IM + ATM 训练方法

### 6.1 Dr. Jonathan Vestal 的认证背景

Dr. Vestal 拥有两项关键认证（在上一轮调研中完全未被提及）：
- **Interactive Metronome (IM) 认证：** 用于增强时序、协调和认知功能
- **Active Therapeutic Motion (ATM) 认证：** 神经肌肉再训练方法

这些认证直接对应了他所使用的训练方法。

### 6.2 ATM（主动治疗性运动）

**什么是 ATM？**
ATM 是一种神经肌肉再训练方法，用于改善功能受损患者的运动模式。它与传统的拉伸/力量训练不同——核心是通过**有意识的、缓慢的、重复的运动**来重新训练神经系统。

**ATM 的核心原则：**
- 做动作时完全保持意识参与
- 动作幅度要小（不需要做到最大范围）
- 重点是**神经系统学习运动的控制**，而不是增加强度
- 配合呼吸进行

**ATM 可应用于茂懿的训练：**

#### ATM 肩颈放松（针对头前倾）

1. 坐直，双手放在膝盖上
2. 非常缓慢向右转头，到感觉颈部有轻微牵拉
3. 停在这个位置，深呼吸3次
4. 再用非常缓慢的速度向左转头
5. 停住，深呼吸3次
6. 重复5轮

#### ATM 骨盆稳定（针对坐姿）

1. 坐直，双手放在骨盆两侧
2. 非常缓慢地将骨盆向前倾斜（找骨盆前倾的感觉）
3. 停住，感受坐骨的压力变化
4. 再缓慢回到中立位
5. 然后向后倾斜（骨盆后倾）
6. 重复8-10次

#### ATM 胸椎伸展（针对驼背坐姿）

1. 坐直，双手抱头
2. 非常缓慢地向后伸展胸椎（不是颈椎！是上背部）
3. 感受胸椎逐个打开
4. 停住，深呼吸2次
5. 缓慢回到起始位置
6. 重复5次

### 6.3 IM + 本体感受训练的融合

Dr. Vestal 的另一个特色是将 IM（节拍）与本体感受训练结合：

1. **闭眼节拍踏步：** 去除了视觉输入，强迫神经系统依靠前庭和本体感受来维持平衡和节奏
2. **不同表面的节拍踏步：** 在硬地板、地毯、枕头等不同表面上做节拍踏步，改变本体感受反馈
3. **负重节拍训练：** 手持小哑铃（或水杯）做节拍踏步，增加运动感觉输入

## 七、第五部分：课程实操演练中的其他关键训练项目

根据课程大纲中的「实操演练内容」模块，以下是应当纳入但上一轮调研缺失的项目：

### 7.1 姿势张力与运动筛查

**课程大纲原文：**「姿势张力与运动筛查」「发育里程碑绘制/对照」

**训练项目：**

#### 高低位交替（Prone Extension / Supine Flexion 切换）

**目的：** 评估和训练姿势性肌肉张力（postural tone）的完整性

**步骤：**
1. 俯卧：双臂向前伸直，抬起双臂和双腿（超人姿势）
2. 保持10-15秒
3. 切换到仰卧：双臂抱膝，下巴贴胸
4. 保持10-15秒
5. 交替3轮

**评估指标：** 在俯卧位时是否能维持双上肢和双下肢离地？身体是否对称？

### 7.2 眼球运动与协调性筛查

**课程大纲原文：**「眼球运动与协调性筛查」

**训练项目（补充上一轮的不足）：**

#### 平滑追踪评估（Dolls Eye 测试的家用版）

1. 让孩子坐着，固定头部
2. 拿一个物品（如笔）在孩子眼前30-40cm处缓慢移动
3. 做水平 H 形、垂直 V 形、圆形、对角线运动
4. 观察：眼睛是否能平滑地跟随？是否有明显的追赶跳（catch-up saccades）？
5. 左右侧是否一致？

#### 视动性眼震（Optokinetic Nystagmus, OKN）训练

**目的：** 训练固定注视时的稳定性（阅读时不跳行）

**方法：**
1. 用条纹纸或OKN鼓（或手机上的OKN视频）
2. 让孩子注视条纹移动
3. 刺激产生正常的视动性眼震（慢速追随+快速回位）
4. 左右方向各30秒
5. 这个测试/训练在功能神经学评估中非常基础，但上一轮完全缺失

### 7.3 交叉模式训练

**课程大纲原文：**「平衡、协调与交叉模式训练」

#### 跨越中线训练（Crossing Midline Drills）

上一轮有「交叉爬行」，但以下变体未覆盖：

1. **墙体交叉模式：**
   - 面对墙站立，离墙一臂距离
   - 右手触摸左侧墙面，左手触摸右侧墙面（交替）
   - 配合节拍器：每拍触一次
   - 这是交叉模式+协调+节拍的组合训练

2. **坐姿交叉触膝：**
   - 坐直
   - 右手触左膝，左手触右膝，交替进行
   - 加速时配合节拍器
   - 可结合认知负荷（触摸时加背诗/计数）

3. **四点跪姿交叉抬起：**
   - 手膝着地
   - 抬起右臂同时抬起左腿（交叉模式）
   - 保持3-5秒，放下
   - 抬起左臂同时抬起右腿
   - 重复10次

### 7.4 用于发育刺激的前庭刺激

**课程大纲原文：**「用于发育刺激的前庭刺激」

#### 前后向加速刺激（比旋转更基础）

1. **直线加速：** 让孩子坐在秋千或滑板车上，前后推送
2. **线性振荡：** 用毯子或床单做摇篮式前后摇摆
3. **垂直加速：** 跳床/蹦床——最有效的前庭刺激方式，上一轮已覆盖

**为什么线性加速重要：**
前庭系统有两个部分：半规管（感知旋转）和耳石器（感知线性加速）。很多ADHD/ASD孩子对旋转的耐受性差，但对线性加速的耐受性更好。**从线性加速开始，逐步过渡到旋转**，是更安全的渐进方案。

## 八、综合训练建议与优先级

### 8.1 优先级排序

基于补充调研发现，以下是需要尽快纳入日常训练的项目（按优先级）：

| 优先级 | 训练项目 | 时间 | 为什么重要 |
|-------|---------|------|-----------|
| ⭐⭐⭐⭐⭐ | **节拍器54BPM踏步** | 每天3-5分钟 | 激活小脑时序功能，改善注意力，有研究直接支持 |
| ⭐⭐⭐⭐⭐ | **Pencil push-ups + Brock string** | 每天3-5分钟 | 解决可能存在的辐辏不足，改善阅读，你已观察到需要 |
| ⭐⭐⭐⭐⭐ | **前庭-眼球运动整合训练** | 每天3-5分钟 | 课程核心实操，上一轮被弱化，VOR训练缺失 |
| ⭐⭐⭐⭐ | **Saccade序列训练** | 每天2-3分钟 | 阅读流畅性的直接决定因素 |
| ⭐⭐⭐⭐ | **ATM胸椎/骨盆训练（你自己+茂懿）** | 每天2-3分钟 | 改善坐姿→改善脑血流→改善注意力 |
| ⭐⭐⭐⭐ | **双手交叉节拍训练** | 每天3分钟 | 结合IM原理的家庭替代，激活全脑 |

### 8.2 每日训练结构建议（15分钟版）

```
早晨（5分钟）：
  1. 冷水拍脸 + 3次深呼吸 —— ✅ 已有
  2. 节拍器54BPM踏步（3分钟）—— ⬅️ **新加入！**
  3. 交叉爬行（2分钟）—— ✅ 已有

白天间歇（5分钟，可分2-3次做）：
  1. Pencil push-ups（2分钟）—— ⬅️ **新加入！**
  2. Saccade练习（1分钟）—— ⬅️ **新加入！**
  3. 海星呼吸（2分钟）—— ✅ 已有

下午/晚上（5分钟）：
  1. 前庭-眼球整合训练（VOR：看固定目标转头，2分钟）—— ⬅️ **新加入！**
  2. 节拍踏步+对侧手臂摆动（3分钟）—— ⬅️ **新加入！**
```

### 8.3 与已有训练的整合提醒

⚠️ **不要替换现有的训练，而是补充。** 已有的「神经功能训练·全家手册」中的训练项目（原始反射整合、前庭训练、迷走神经激活、精细运动训练）都有科学依据。本文件补充的是该手册中遗漏的项目。

**总体原则：**
- 每天总训练时间保持在20-25分钟（不要超过30分钟）
- 用补充项目替换掉手册中效果相似但强度较低的项目
- 保留手册中独特的项目（海星呼吸、蜜蜂呼吸、精细指尖对碰等）
- 新增训练至少持续4周再评估效果
